# 1 課題名
（仮）ママめし（CtoCの家庭料理マーケットプレイス）

# 2 課題内容（どんな内容か）
地域のベテラン主婦（ママ）が作る、手料理を、忙しく働く現役世代へ届けるサービス。

以下着想に至った背景
- 私は一人暮らしで自炊ゼロ。毎日松屋とコンビニ飯でご飯が「エサ化」。正月に実家に帰った際、実家のご飯・手料理はやはり美味しいと感動、と同時に。他の人はどんなご飯を食べているのか気になった。
- 「隣の家の晩御飯は隣の家の人しか知らない」というのが現状。デジタルの力を使えばこの埋蔵資産的なものを解放できるのでは？
- 和食は無形文化遺産に登録されたけど、その味を受け継いでいる人ってごく僅かだよね。レストランで提供される和食もいいけど、各家庭で作られる家庭料理こそ真の和食では？和食の保存という文脈でも良さそうだな。日本の和食のプロであるシニア主婦層を最高のクリエイター集団として世の舞台に出したいな。
- 正直AIに疲れてきたな。今後AI疲れの波がきて、人間として、人との繋がりを求めてC2Cのマーケットプレイスに回帰する動きが出てくるんじゃないかな。アメリカのVCもAIエージェントばっかりに投資してるな。AI時代のアンチテーゼ的な事業（=効率の対極にある手間だったり、「誰かが自分のために作ってくれた」という情緒的価値を表現した事業）が最高なのでは？
